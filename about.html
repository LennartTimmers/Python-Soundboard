About the soundboard

This soundboard is the result of a personal pet project. I've been watching Broken Bonds since the beginning of the campaign and have thoroughly enjoyed it so far. After P'mis transformed and his voice changed I suddenly had the idea to create a soundboard with some of the hilarious remarks and bits and pieces of voice acting.


There are of course still a lot of nice audio clips missing but sadly it's too much work for me to find and convert each funny remark to a small audio file. If you have any sound files which you want me to add you can email me and I'll try to add the sound to the soundboard!


If you are curious about the technical side of things then you can checkout the code right here. The site works with a simple html file containing some javascript. This javascript sends a request to the API which is built using Flask in python. The API then returns a list of all available soundclips which the javascript loops over and dynamically adds the soundsfiles to each character.


If you want to make your own soundboard you can use my template which you can find right here.
